## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run dev

# build
npm run build
```

## Test

```bash

npm run test

# watch mode
npm run test:watch

# test coverage
npm run test:coverage
```

## Linters

```bash
# lint
npm run lint
```
