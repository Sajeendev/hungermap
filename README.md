# Hungermap

## Description

Hungermap project.
Add your API URL to api.constants.ts to make it work.

## Technologies

- NextJs 15
- Mantine UI

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run dev

# build
npm run build
```

## Test

```bash
# unit tests
npm run test

# watch mode
npm run test:watch

# test coverage
npm run test:coverage
```

## Linters

```bash
# lint
npm run lint

# format
npm run format
```
