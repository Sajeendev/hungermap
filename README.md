# Hungermap

## Description

Hungermap: WFP Assessment

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run dev

# build
npm run build
```

## Test

```bash
# unit tests
npm run test

# watch mode
npm run test:watch

# test coverage
npm run test:coverage
```

## Linters

```bash
# check types
npm run check:types

# lint
npm run lint

# format
npm run format
```
